В Python существует понятие классов и их объектов, и первым шагом вам необходимо осознать, что в Python всё является объектом какого-то класса. Например:
int (Целое число)	42, -7, 0, 1000
float (Вещественное число)	3.14, -0.5, 2.0, 1.618
str (Строка)	"Привет", "Как дела?"
list (Список)	[1, 2, 3], ["a", "b"], [], [10, "текст", 3.14]

Класс — это небольшая программа, в которой с помощью кода описаны свойства и поведение объекта. Например, есть класс int (целые числа), в нём с помощью кода указано, что целые числа можно складывать, вычитать и так далее. Каждое целое число является объектом этого класса.
С числами мы можем делать только конкретные действия, указанные в классе. Например, мы можем сложить 1 + 1, потому что в классе это указано с помощью кода. Но если, например, сделать вот так: 1 _ 1, то Python вызовет ошибку, потому что в классе нет информации о таком действии с целыми числами.

Целые числа (int) — это числа без дробной части. Примеры: 1, -5, 100.
Вещественные числа (float) — это числа с дробной частью. Примеры: 3.14, -0.5, 2.0.
И те, и другие могут быть как положительными, так и отрицательными. Чаще всего их используют для математических вычислений.

Важное правило для Python: В вещественных числах целую часть от дробной всегда отделяют точкой, а не запятой. Так 2.5 — правильно, а 2,5 вызовет ошибку.

✅ Правильно: 3.14, -0.5, 10.0
❌ Ошибка: 3,14, -0,5, 10,0

Строка (str) — это тип данных для работы с текстом. В Python строку всегда заключают в кавычки.
