# TkinterLearningL002

Что такое Tkinter?
Tkinter – это пакет для Python, предназначенный для работы с библиотекой Tk. Библиотека Tk содержит компоненты графического интерфейса пользователя (graphical user interface – GUI). Эта библиотека написана на языке программирования Tcl.

Tkinter нужен, чтобы делать программы с окнами, кнопками, ПОЛЯМИ ввода и текстовыми метками — то есть с тем интерфейсом, к которому мы привыкли в обычных приложениях. Без него Python-программы работают только в консоли: ввод и вывод через терминал. С Tkinter можно создавать калькуляторы, часы, простые игры, формы для заполнения данных — всё, что пользователь видит как отдельное окно с элементами, на которые Можно нажимать.

Виджеты — это строительные блоки интерфейса: кнопки (Button), текстовые метки (Label), поля для ввода (Entry), чекбоксы, ползунки и т.д. Каждый виджет — это объект Python, который создаётся, настраивается (текст, цвет, размер), а затем размещается в окне с помощью методов .pack(), .grid() или .place().

Виджет Button — интерактивный элемент, который выполняет функцию (коллбэк) при нажатии.
