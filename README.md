В Python существует понятие классов и их объектов, и первым шагом вам необходимо осознать, что в Python всё является объектом какого-то класса. Например:
int (Целое число)	42, -7, 0, 1000
float (Вещественное число)	3.14, -0.5, 2.0, 1.618
str (Строка)	"Привет", "Как дела?"
list (Список)	[1, 2, 3], ["a", "b"], [], [10, "текст", 3.14]

Класс — это небольшая программа, в которой с помощью кода описаны свойства и поведение объекта. Например, есть класс int (целые числа), в нём с помощью кода указано, что целые числа можно складывать, вычитать и так далее. Каждое целое число является объектом этого класса.
